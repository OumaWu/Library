# Library
Library management system reimplemented with JavaFX as front-end

Frontend solution : 
  - JavaFX(MVC)

Backend solutions :
  - Database : MySQL
  - Java

Design Patterns :
  - Singleton Pattern
  - Proxy Pattern
  - Factory Pattern (BookFactory)
  - Façade Pattern (DBTool)
  - Strategy Pattern (View for Novel - Manuel - Magazine)
  - MVC
